# Ravenous

## Purpose
Ravenous is a React-based restaurant recommendation app inspired by Yelp. Users can search for local businesses by location and keywords.

## Technologies Used
- React (Vite)
- JavaScript
- HTML & CSS
- Git & GitHub

## Features
- Search bar for business name and location
- Scrollable list of mock restaurant data
- Responsive layout
